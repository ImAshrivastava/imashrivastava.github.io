Major Projects



2FAB Authenticator

Designed and developed “Authenticator: Two-Factor Biometric Authentication Desktop Application” in processing.python(front-end) and python (backend) communicating over socket communication, to perform fingerprint matching using KNearest Neighbour(KNN) and handwritten signature matching using Structural Similarity Index(SSIM) for verifying individual identity.

eSignature: Handwritten Signature Acquisition Application

Designed and developed “eSignature” in processing.python for acquiring handwritten signatures using display tablets.

FIST: Fingerprint Imaging Software Technology

Designed and developed “FIST: Fingerprint Imaging Software Technology”, a cross-platform desktop application in processing.python (front-end) and python (backend) communicating over socket communication, to implement serial communication between desktop application and biometric stylus for activating I/O functionality of ID809 capacitive fingerprint sensor with complete or restricted control over GET-SET parameters including fingerprint template extraction and fingerprint image acquisition function.

2FAB Stylus

Integrated ID809 fingerprint sensor and RP2040 microcontroller in a stylus for concurrently acquiring fingerprint image and handwritten signature image drawn on graphic display tablet for Two-Factor Biometric Authentication System.

Aerial Bomb/Aerial Missile

Fabricated low-intensity, air-to-surface missile and aerial bomb made using gunpowder as the primary explosive component ignited at specific altitude or by using wireless command system for the purpose of disintegration of riots or remote execution using unmanned combat aerial vehicle.

Paper-based Biosensing System for Detection of Mastitis in Dairy Cattle

Devised a novel low-cost paper-based biosensor to detect increase in electrical conductance and count of somatic cells in milk using bibulous paper and current sensor for prevention of mastitis in dairy cattle.

iThink: An Electroencephalography-based Brain Computer Interface

Proposed a novel algorithm and trained a neural network to process EEG raw data for real-time brain-to-text, brain-tospeech and brain-to-command using wearable brain-computer interface.

P300 Speller for Brain Computer Interface

Devised a brain-computer interface to interpret visually evoked p300 signal using support vector machine for spelling words in a custom word processor.

AudioOptic: A Computer Vision-based Wearable Visual Aid for Visually Challenged

Embedded array of optical sensors and processors in a headgear to implement a wearable visual aid working on parallel computing to detect, recognize and track objects, faces and text to translate them into speech for navigation and real-time application using concurrent visual processing algorithms.

MedAssist: An Automated Medicine Dispenser

Developed an Internet of Things-based automated medicine dispenser system using Zigbee network for personal use and for hospitals.

Unmanned Ground Vehicle with Vision for Remote Security and Surveillance

Implemented a continuous tracked, semi-autonomous unmanned ground vehicle with six degrees of freedom robotics arm for remote security and surveillance.
