    iThink: An Electroencephalogram-based Brain Computer Interface
    for Brain to Text, Brain to Speech and Brain to Command


    Texas Instruments Laboratory, Centre of Excellence on Commbedded Systems
    Department of Electronics and Communication Engineering
    Visvesvaraya National Institute of Technology

    ABSTRACT

    iThink is an electroencephalogram-based non-invasive, wearable, peripheral brain-computer interface for direct communication of the human brain and computer software. It allows humans to think and type text based on a controlled and focused cognition process. The interface is used as an input device for computer software, word processors and other such services by articulating alphabets in an attentive state naturally or in the presence of a visual stimulus. The device enables users, including patients with disabilities and disorders such as paralysis or amyotrophic lateral sclerosis, to communicate by typing in a custom word processor. The stream of incoming EEG data is processed through a neural network with a simultaneous character encoding technique compared with trained data labeled for different cognitive states and attention levels. The interface runs on a system-on-chip computer that produces natural speech through audio output using a TTS engine every time a user successfully inputs an alphabet. The device is integrated with word and sentence completion algorithms, which recognize alphabetical order from an embedded database of words, sentences and phrases to generate real-time recommendations as the user generates sequences of alphabets. The interface has shown limited accuracy with sufficient test conditions possible with sample data available. The current system is a research prototype only and will require significant development before it could be deployed in real-life settings.










    Audio Optic: A Computer Vision-based Wearable Visual Aid


    IKSC Knowledge Bridge and the Blind Relief Association

    ABSTRACT

    Audio Optic is a computer vision-based dynamic wearable visual aid device to support the visually impaired for environment interaction using an audio and haptic feedback mechanism. The device communicates with the user in the form of natural speech representing their visual environment. The environment is captured using an array of three cameras to detect, recognize and track objects, faces and text to translate them into speech in real-time. The visually impaired hears the set of instructions using headphones and receives vibrations using micro-motors for navigation. The device runs on a pair of system-on-chip computers concurrently and simultaneously computing and processing visual data. The algorithm computes the frames of data based on predefined sets of trained data. The stream of data undergoes mathematical transformation for image and video processing through a neural network. The system compares unknown environment entities calling cloud computing services to find the possible matches for recognition. The device uses integrated ultrasonic sensors to continuously track nearby moments. The device uses IR sensors to expand the usability for the night time. The parallel computing architecture achieved an accuracy of 87%, enabling visually challenged to perform sophisticated tasks with minimum guidance.
